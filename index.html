Step 1: Install 7-Zip

First, you need to download 7-Zip, because it is required to uncompress the Dolphin installation. Click the 7-Zip link found in the Download Links section to the left. Once you reach the webpage, install the corresponding version for you device, specifically, I would recommend dowloading the ones on the top, and not alpha releases. Once the download is complete, open the file, and click through the prompts to finish the installation.

Step 2: Install Visual Studio C++ 2019

Next, you'll need to install Visual Studio C++ 2019, because the version of Dolphin you're installing, the beta, needs this to run. To begin, open the Visual Studio 2019 link, found in the Download Links section. Next, on the new tab, scroll down the page to the "Visual Studio 2015, 2017 and 2019" section. Download the corresponding version to your device. Just like 7-Zip, open the file, and click through the prompts to install it.

Step 3: Install Dolphin

Now that you have installed 7-Zip and Visual Studio, you can finally install Dolphin. Open the Dolphin Emulator link in the Download Links section. Next, on the Dolphin website, download the latest Beta version on the page for your corresponding device. Once the download finishes, you will have a file with a .7z file extension on this. Open up 7-Zip, click file, then open, and select the Dolphin file. It will open, showing you the contents. Click extract, and select a folder you want your installation to stay in.

Step 4: Home Menu Wad download

Now, your dolphin installation is in working order. However, there are still many improvements to be made. First, we are adding the home menu to dolphin. To start, we must install NUS Downloader. Go to the NUS Downloader link in the Download Links section, then on that page, go to the box on the top right, and click the download link in that box. Next, unzip the file, open the unzipped folder, and open the "NUS Downloader.exe". If you're like me, you might get a popup about installing the .NET FRAMEWORK, and if so, follow the prompts to install it, and reopen the exe. Once you open the application, click the Database button in the top left. In the dropdown, click "System", and then click "0000000 100000002 - System Menu", and finally, scroll down to the bottom of the next drop down. Click the latest version for your region (E=Europe, J=Japan, U=USA, K=Korea). After this, tick the "Pack Wad" box. Finally, click the "Start NUS Downlad!" near the top.

Step 5: Home Menu Wad Installation

Now that you've downloaded the wad, you need to install it. First, open Dolphin, and click the Tools button on the top. Next, in the dropdown, click Install Was. Next, a file explorer window should pop up. Locate and open the unzipped folder, then into the "titles" folder. Next, open the "0000000100000002" folder, then the folder with the corresponding version number you downloaded. Finally open the RVL-WiiSystemmenu wad. In seconds, you should get a popup saying that it successfuly installed. Once that's done, you need to click the Tools button again, but this time, click "Perform Online System Update", and then current region. Click yes on the popup about connecting to wifi, and then wait for it to install. If all goes well, you should get a popup saying that the download was completed successfuly.
